Information

This server hosts a fairly sizable number of Dutch regulations in the CEN MetaLex XML and RDF formats. Current coverage is in the order of 8k documents, about 30% of all regulations hosted at http://overheid.wetten.nl.

The regulations were automatically converted by means of a (hopefully) generic conversion script, that is able to perform a similar conversion for any 'legislative' XML based on a standard mapping from legacy XML elements to CEN MetaLex. See the script's GitHub page for installation and usage instructions.

We hope to be able to provide on-demand conversion of documents not currently available in the repository

Example: "Wet werk en inkomen naar arbeidsvermogen"

Technical Details

RDF representations of regulations reside in a ClioPatria SWI-Prolog Semantic Web Server, developed at VU University Amsterdam and the University of Amsterdam.

MetaLex XML documents currently reside on the filesystem of the server. We plan to move these to a native XML database to support XQuery access to their structure.
